# ⚡ Быстрый гайд: Группы студентов в рабочих программах

## 🎯 В одной строке
Теперь при создании рабочей программы вы можете выбрать группу студентов колледжа, к которой относится программа.

---

## 🚀 Три шага к использованию

### Шаг 1️⃣: Создание программы
```
Откройте: Рабочие программы → + Создать новую программу
```

### Шаг 2️⃣: Выбор группы
```
В форме найдите поле "Группа студентов"
Выберите из выпадающего списка (5 А, 5 Б и т.д.)
```

### Шаг 3️⃣: Сохранение
```
Нажмите кнопку "Сохранить программу"
✅ Готово! Программа привязана к группе
```

---

## 👀 Где видно группу?

### В списке программ:
| Название | Предмет | Группа |
|----------|---------|--------|
| Математика | Математика | 🟢 5 А |
| История | История | 🟢 5 Б |

### При редактировании:
```
Рабочие программы → Редакт. → поле "Группа студентов"
```

## ❓ Часто задаваемые вопросы

**В: Обязательно ли выбирать группу?**
О: Нет, это опционально. Можете оставить пусто, если программа для всех групп.

**В: Могу ли я изменить группу потом?**
О: Да, просто отредактируйте программу и выберите другую группу.

**В: Что если удалить группу?**
О: Программа останется, но поле будет пусто. Можно переназначить другой группе.

**В: Это влияет на оценки?**
О: Пока что нет, но это может быть полезно для будущих функций.

## 💡 Полезные советы

✅ **Добавляйте группу при создании** — это облегчит управление потом

✅ **Используйте для организации** — группируйте программы по группам студентов

✅ **Оставляйте пусто для общих программ** — если программа для нескольких групп

✅ **Проверяйте в списке** — убедитесь, что группа выбрана правильно
