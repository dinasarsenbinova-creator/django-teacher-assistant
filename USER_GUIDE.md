# 📚 Ассистент педагога - Краткое руководство пользователя

## 🎯 Быстрый старт (5 минут)

### Шаг 1: Запуск приложения
```powershell
# Откройте PowerShell в папке проекта
cd p:\workspace\django_site

# Активируйте виртуальное окружение
.\.venv\Scripts\Activate.ps1

# Запустите сервер
python manage.py runserver
```

### Шаг 2: Вход в приложение
1. Откройте браузер и перейдите на: **http://localhost:8000/teacher/**
2. Введите учетные данные:
   - **Логин**: teacher
   - **Пароль**: password123
3. Вы увидите главную панель управления

---

## 📊 Основные модули

### 1️⃣ Панель управления (Dashboard)
**Что здесь находится:**
- Быстрый просмотр статистики
- Предстоящие уроки на неделю
- Расписание на сегодня
- Последние добавленные оценки и тесты
- Кнопки быстрого доступа для добавления данных

**Как использовать:**
- Смотрите на главной странице информацию о своей работе
- Используйте кнопки "Быстрые действия" для быстрого добавления данных

---

### 2️⃣ Рабочие программы (📚)
**Что здесь находится:**
- Ваши рабочие программы по предметам
- Темы в каждой программе
- Связанные уроки и тесты

**Как использовать:**

**Создание программы:**
1. Нажмите "+ Создать новую программу"
2. Заполните форму:
   - **Название**: "Математика"
   - **Предмет**: "Математика"
   - **Класс**: выберите из списка
   - **Учебный год**: 2025, 2026 и т.д.
3. Нажмите "Сохранить"

**Просмотр программы:**
1. Нажмите на программу в списке
2. Видите все темы, уроки и тесты этой программы

---

### 3️⃣ Расписание (📅)
**Что здесь находится:**
- Еженедельное расписание занятий
- Время, дни недели, кабинеты

**Как использовать:**

**Добавление занятия:**
1. Нажмите "+ Добавить занятие"
2. Заполните форму:
   - **День недели**: выберите день
   - **Время начала/конца**: укажите время
   - **Предмет**: математика, русский язык и т.д.
   - **Класс**: 5А, 5Б и т.д.
   - **Кабинет**: номер кабинета (опционально)
3. Нажмите "Добавить"

**Просмотр расписания:**
- Таблица показывает все занятия по дням недели
- На главной странице видны занятия на сегодня

---

### 4️⃣ Планы уроков (✏️)
**Что здесь находится:**
- Подробные планы всех уроков
- Цели, содержание, материалы
- Домашние задания

**Как использовать:**

**Создание плана урока:**
1. Нажмите "+ Создать урок"
2. Заполните форму:
   - **Рабочая программа**: выберите программу (например, Математика)
   - **Тема**: выберите тему из программы
   - **Название**: "Натуральные числа"
   - **Дата**: выберите дату урока
   - **Класс**: 5А
   - **Предмет**: Математика
3. Добавьте содержание:
   - **Цели и задачи**: что будут учиться ученики
   - **Содержание урока**: основной материал
   - **Материалы**: книги, ссылки, раздаточные материалы
   - **Домашнее задание**: что задать на дом
4. Нажмите "Сохранить"

**Просмотр и редактирование:**
- Нажмите "Просмотр" чтобы увидеть полный план
- Нажмите "Редакт." чтобы изменить план

---

### 5️⃣ Журнал оценок (📝)
**Что здесь находится:**
- Все оценки учащихся
- Тип работы (домашнее, классная, контрольная, экзамен)
- Дата и комментарии

**Как использовать:**

**Добавление оценки:**
1. Нажмите "+ Добавить оценку"
2. Заполните форму:
   - **ФИО ученика**: напишите полное имя (например, "Иван Петров")
   - **Класс**: 5А
   - **Предмет**: Математика
   - **Оценка**: выберите от 1 до 5
   - **Тип работы**: 
     * Домашнее задание
     * Классная работа
     * Контрольная работа
     * Экзамен
   - **Комментарий**: опционально, например "Хорошая работа"
3. Нажмите "Добавить"

**Просмотр оценок:**
1. Видите список всех оценок
2. Используйте фильтры для поиска:
   - По классу (5А, 5Б и т.д.)
   - По предмету (Математика, Русский язык и т.д.)

---

### 6️⃣ Тесты и контрольные работы (📋)
**Что здесь находится:**
- Созданные тесты и контрольные работы
- Вопросы и задания
- Результаты тестирования учащихся

**Как использовать:**

**Создание теста:**
1. Нажмите "+ Создать новый тест"
2. Заполните форму:
   - **Название**: "Тест: Натуральные числа"
   - **Тип**: Тест / Контрольная работа / Викторина / Экзамен
   - **Рабочая программа**: выберите программу
   - **Тема**: выберите тему
   - **Предмет**: Математика
   - **Класс**: 5А
   - **Содержание**: напишите вопросы/задания (каждое с новой строки)
   - **Ключ ответов**: напишите правильные ответы
   - **Макс. балл**: 50 или 100
   - **Продолжительность**: время в минутах
3. Нажмите "Сохранить"

**Просмотр теста:**
1. Нажмите на тест
2. Видите:
   - Содержание теста
   - Ключ ответов
   - Результаты учащихся

## ❓ Часто задаваемые вопросы

**Q: Как изменить пароль?**  
A: Пока функция не реализована. Используйте админ-интерфейс:
http://localhost:8000/admin/ → Users → teacher

**Q: Можно ли добавить несколько педагогов?**  
A: Да, в админ-интерфейсе: http://localhost:8000/admin/ → Users → Add user

**Q: Как сделать резервную копию данных?**  
A: Скопируйте файл `db.sqlite3`

**Q: Как удалить все данные и начать заново?**  
A: Удалите файл `db.sqlite3` и переинициализируйте:
```
python manage.py migrate
python load_test_data.py
```

**Q: Как получить доступ к админ-интерфейсу?**  
A: http://localhost:8000/admin/
Создайте суперпользователя: `python manage.py createsuperuser`
