# Ассистент педагога 🎓

Полнофункциональное веб-приложение для управления профессиональной деятельностью учителя. Система помогает педагогам организовать свою работу, управлять расписанием, вести журнал оценок, планировать уроки и создавать тесты.

## ✨ Основные функции

- 📚 **Рабочие программы** - Управление учебными программами и темами
- 📅 **Расписание** - Ведение еженедельного расписания занятий
- ✏️ **Планы уроков** - Подробное планирование каждого урока
- 📝 **Журнал оценок** - Ведение оценок учащихся с фильтрацией
- 📋 **Тесты и контрольные работы** - Создание и сохранение тестов с результатами

## 🚀 Быстрый старт

### 1️⃣ Windows (PowerShell)

```powershell
# Активация виртуального окружения
.\.venv\Scripts\Activate.ps1

# Установка зависимостей
pip install -r requirements.txt

# Применение миграций
python manage.py migrate

# Загрузка тестовых данных
python load_test_data.py

# Запуск сервера
python manage.py runserver
```

Затем откройте в браузере: **http://localhost:8000/teacher/**

### 2️⃣ Windows (Batch файл)

```bash
# Просто запустите
setup.bat
```

### 3️⃣ Linux/Mac

```bash
# Активация виртуального окружения
source .venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt

# Применение миграций
python manage.py migrate

# Загрузка тестовых данных
python load_test_data.py

# Запуск сервера
python manage.py runserver
```

## 📊 Основные модели

### Curriculum (Рабочая программа)
- Название, предмет, класс, год
- Автоматическая привязка уроков и тестов

### Topic (Тема)
- Название, описание, часы
- Связь с рабочей программой

### Schedule (Расписание)
- День недели, время, кабинет
- Быстрый просмотр на день

### Lesson (Урок)
- Цели, содержание, материалы
- Домашнее задание и примечания
- Связь с темой и программой

### Grade (Оценка)
- ФИО ученика, оценка (1-5)
- Тип работы, комментарий
- Фильтрация по классам и предметам

### Test (Тест)
- Содержание, ключ ответов
- Максимальный балл, время
- Отслеживание результатов
