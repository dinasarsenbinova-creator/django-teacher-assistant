## 🐛 ОТЛАДКА

Если проблема сохраняется:

### 1. Проверьте консоль браузера
1. Откройте страницу викторины
2. Нажмите **F12** (открывает DevTools)
3. Перейдите на вкладку "Console"
4. Обновите страницу
5. Проверьте наличие ошибок JavaScript

### 2. Проверьте исходный код страницы
1. Откройте страницу викторины
2. Нажмите **Ctrl + U** (View Source)
3. Найдите (Ctrl+F): "DEBUG INFO"
4. Проверьте, что options_len показывает > 0

### 3. Проверьте серверные логи
1. Откройте терминал где запущен Django сервер
2. При загрузке страницы викторины должны появиться отладочные сообщения:
   ```
   === QUIZ LOADING DEBUG ===
   Quiz ID: 8, Title: Викторина: Системы счисления
   Questions count: 15
   First question ID: 46
   First question text: Сколько байт в одном кибибайте?
   First question type: multiple_choice
   First question options: ['1240', '1000', '2400', '1024']
   First question options type: <class 'list'>
   ========================
   ```

### 4. Запустите тестовые скрипты
```bash
# Проверка викторин и вопросов
python check_quiz.py

# Детальная проверка
python check_quiz_detailed.py
```
