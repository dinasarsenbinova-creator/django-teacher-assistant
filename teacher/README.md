# Ассистент педагога - Полное руководство

## Описание

Ассистент педагога - это полнофункциональное веб-приложение на Django для управления профессиональной деятельностью учителя. Система помогает педагогам организовать свою работу, управлять расписанием, вести журнал оценок, планировать уроки и создавать тесты.

## Основные функции

### 📚 Рабочие программы
- Создание и управление рабочими программами
- Организация учебного материала по темам
- Привязка уроков и тестов к программам
- Выбор тем для загрузки в электронный журнал

### 📅 Расписание
- Управление еженедельным расписанием занятий
- Указание кабинетов и времени занятий
- Быстрый просмотр расписания на день

### ✏️ Планы уроков
- Детальное планирование каждого урока
- Указание целей, содержания, материалов
- Добавление домашних заданий
- Привязка к темам рабочей программы

### 📝 Журнал оценок
- Ведение оценок учащихся
- Фильтрация по классам и предметам
- Указание типа работы (домашнее, классная, контрольная, экзамен)
- Комментарии к оценкам

### 📋 Тесты и контрольные работы
- Создание и хранение тестов
- Сохранение ключей ответов
- Отслеживание результатов тестирования
- Расчет процента выполнения

## Установка и настройка

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Применение миграций
```bash
python manage.py migrate
```

### 3. Создание суперпользователя
```bash
python manage.py createsuperuser
```

### 4. Запуск сервера
```bash
python manage.py runserver
```

Приложение доступно по адресу: `http://localhost:8000`

## Использование

### Доступ к приложению
1. Перейдите на http://localhost:8000/teacher/
2. Пройдите аутентификацию (используйте учетные данные суперпользователя)

### Панель управления
Главная страница показывает:
- Предстоящие уроки
- Расписание на сегодня
- Активные рабочие программы
- Свежие оценки
- Последние тесты

### Работа с рабочими программами
1. Перейдите в раздел "Рабочие программы"
2. Создайте новую программу указав:
   - Название программы
   - Предмет
   - Класс
   - Учебный год

### Создание плана урока
1. Перейдите в раздел "Планы уроков"
2. Нажмите "Создать урок"
3. Заполните форму:
   - Выберите рабочую программу и тему
   - Укажите дату урока
   - Добавьте цели и содержание
   - Опишите материалы и домашнее задание

### Ведение журнала оценок
1. Перейдите в "Журнал оценок"
2. Нажмите "Добавить оценку"
3. Заполните форму:
   - ФИО ученика и класс
   - Оценка (1-5)
   - Тип работы (домашнее, классная, контрольная, экзамен)
   - Добавьте комментарий если нужно

### Создание тестов
1. Перейдите в "Тесты и контрольные работы"
2. Нажмите "Создать новый тест"
3. Заполните форму:
   - Название теста
   - Тип (тест, контрольная работа, викторина, экзамен)
   - Содержание (вопросы/задания)
   - Ключ ответов
   - Максимальный балл
   - Продолжительность

## Модели данных

### Curriculum (Рабочая программа)
- Название
- Предмет
- Класс
- Учебный год
- Дата создания/обновления

### Topic (Тема)
- Название темы
- Описание
- Порядок в программе
- Количество часов
- Связь с программой

### Schedule (Расписание)
- Предмет и класс
- День недели
- Время начала/окончания
- Номер кабинета
- Педагог

### Lesson (Урок)
- Название и дата
- Класс и предмет
- Цели и задачи
- Содержание
- Материалы и ресурсы
- Домашнее задание
- Примечания
- Связь с программой и темой

### Grade (Оценка)
- ФИО ученика и класс
- Предмет
- Оценка (1-5)
- Тип работы
- Комментарий
- Дата оценки

### Test (Тест)
- Название и описание
- Тип (тест, контрольная, экзамен и т.д.)
- Содержание (вопросы)
- Ключ ответов
- Максимальный балл
- Продолжительность
- Класс и предмет

### TestResult (Результат теста)
- Студент
- Набранный балл
- Процент выполнения
- Комментарии
- Дата сдачи
