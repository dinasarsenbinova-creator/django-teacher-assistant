# ✨ Функция: Связь групп студентов с рабочими программами

## 📋 Описание

Добавлена возможность связывать группы студентов колледжа непосредственно при создании или редактировании рабочей программы. Это позволяет:

- **Быстро определить целевую аудиторию** программы
- **Организовать программы по группам** студентов
- **Улучшить отчётность** и анализ по группам
- **Связать программы с расписанием** групп

## 🎯 Использование

### Создание новой программы с группой

1. Перейдите на **Рабочие программы** → **+ Создать новую программу**
2. Заполните основные поля:
   - **Название программы** (обязательно)
   - **Предмет** (опционально)
   - **Класс** (обязательно)
   - **Группа студентов** ← **НОВОЕ ПОЛЕ**
   - **Учебный год** (обязательно)

3. В поле "Группа студентов" выберите одну из существующих групп (например, "5 А", "5 Б")
4. Нажмите **Сохранить программу**

### Просмотр программ по группам

На странице **Рабочие программы** теперь отображается столбец "Группа студентов":

```
Название              | Предмет      | Класс  | Группа студентов | Год
Математика 5 класс   | Математика   | 5      | 5 А              | 2025
История России       | История      | 5      | 5 Б              | 2025
```

### Редактирование связи группы

Чтобы изменить или добавить группу студентов:

1. Откройте программу через кнопку **Редакт.**
2. В поле "Группа студентов" выберите другую группу (или оставьте пустым)
3. Нажмите **Сохранить программу**

## 🗄️ Технические подробности

### Изменения в модели

**Файл:** `teacher/models.py`

Добавлено поле в модель `Curriculum`:

```python
student_group = models.ForeignKey(
    "StudentGroup",
    on_delete=models.SET_NULL,
    null=True,
    blank=True,
    related_name="curriculums",
    verbose_name="Группа студентов",
)
```

**Особенности:**
- Поле опционально (`null=True, blank=True`)
- Связано с моделью `StudentGroup` через ForeignKey
- При удалении группы, связь программы обнуляется (`SET_NULL`)
- Имеет обратную связь `curriculums` для доступа из StudentGroup

### Изменения в представлениях

## 📚 Примеры использования

### Пример 1: Создание программы для конкретной группы

```
Название: Математика 5 класс
Предмет: Математика
Класс: 5
Группа студентов: 5 А
Год: 2025
```

Результат: Программа автоматически связана с группой "5 А"

### Пример 2: Программа без группы (общая для нескольких групп)

```
Название: История общего курса
Предмет: История
Класс: 5
Группа студентов: (оставить пустым)
Год: 2025
```

Результат: Программа доступна для всех групп

### Пример 3: Изменение группы для существующей программы

Отредактируйте существующую программу и выберите другую группу.

## 🔗 Связанные функции

- **Расписание** — может использовать информацию о группе из программы
- **Оценки** — связаны через group_name (в будущем можно улучшить)
- **Нейросеть** — может обучаться на данных конкретной группы

## ⚙️ Интеграция с другими компонентами

### Если нужно получить все программы для группы:

```python
from teacher.models import StudentGroup, Curriculum

# Получить группу
group = StudentGroup.objects.get(name="5 А")

# Получить все программы для этой группы
programs = group.curriculums.all()
```

### Если нужно создать программу для группы программно:

```python
from teacher.models import Curriculum, StudentGroup, Subject

group = StudentGroup.objects.get(name="5 А")
subject = Subject.objects.get(name="Математика")

curriculum = Curriculum.objects.create(
    title="Математика 5 класс",
    subject=subject,
    class_level="5",
    student_group=group,
    year=2025
)
```

## 📊 Админ-панель

Поле "Группа студентов" автоматически отображается в админ-панели Django для модели `Curriculum`:

1. Перейдите на **http://localhost:8000/admin/teacher/curriculum/**
2. Создайте или отредактируйте программу
3. В поле "Группа студентов" выберите нужную группу

## 🎓 Лучшие практики

1. **Привязывайте группы при создании** — это облегчает управление позже
2. **Используйте стандартные имена групп** — "5 А", "5 Б", т.д. для консистентности
3. **Оставляйте пусто для общих программ** — если программа применима ко всем группам
4. **Проверяйте в списке программ** — убедитесь, что группа отображается корректно

## ❓ Вопросы и ответы

**В: Что произойдёт, если удалить группу?**
О: Программа останется, но поле "Группа студентов" будет пусто. Программу можно переназначить другой группе.

**В: Могу ли я иметь одну программу для нескольких групп?**
О: Текущая версия поддерживает связь один-к-одному. Для множественной связи создайте отдельные копии программы.

**В: Как это влияет на оценки и тесты?**
О: Пока что не влияет напрямую, но в будущем можно использовать эту информацию для более точного анализа по группам.
