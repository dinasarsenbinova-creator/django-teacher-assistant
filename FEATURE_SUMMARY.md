# 🎯 Сводка по добавленной функции

## Что было добавлено?

**Возможность связывать группы студентов при создании рабочих программ**

## Где это видно?

### 1️⃣ При создании программы
- Откройте: **Рабочие программы** → **+ Создать новую программу**
- Новое поле: **"Группа студентов"** 
- Выберите одну из существующих групп (5 А, 5 Б, и т.д.)

### 2️⃣ В списке программ
- Теперь в таблице есть столбец **"Группа студентов"**
- Зелёный значок с именем группы

### 3️⃣ При редактировании
- Нажмите **"Редакт."** на программе
- Измените или добавьте группу студентов

## 🚀 Как использовать?

1. **Создайте новую программу**
   - Откройте форму создания
   - Заполните все поля
   - Выберите группу из выпадающего списка
   - Сохраните

2. **Отредактируйте существующую**
   - Нажмите "Редакт." на программе
   - Измените группу
   - Сохраните

3. **Создавайте программы без группы**
   - Просто не выбирайте группу (оставьте пусто)
   - Программа будет доступна всем
